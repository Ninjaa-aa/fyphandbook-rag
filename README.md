# FYP Handbook RAG Assistant

Text-only Retrieval-Augmented Generation (RAG) assistant for the FAST-NUCES *BS Final Year Project Handbook 2023*. It answers student questions from the handbook only, cites source pages inline, and refuses out-of-scope questions with the assignment-required message.

## Architecture

- **Corpus:** single handbook PDF at `docs/3. FYP-Handbook-2023.pdf`
- **Parser:** PyMuPDF page extraction with printed page-number metadata
- **Chunker:** heading-aware section chunks plus page fallback chunks
- **Chunk settings:** `CHUNK_SIZE = 500`, `CHUNK_OVERLAP = 100`
- **Dense embeddings:** `BAAI/bge-m3` through `sentence-transformers`
- **Vector DB:** persistent ChromaDB cosine index in `data/chroma`
- **Sparse retrieval:** BM25 with `rank_bm25`, persisted to `data/bm25.pkl`
- **Hybrid retrieval:** Reciprocal Rank Fusion (RRF, `RRF_K = 60`)
- **Reranker:** `BAAI/bge-reranker-v2-m3` cross-encoder
- **Diversity:** MMR with `MMR_LAMBDA = 0.7`
- **LLM:** Groq `llama-3.3-70b-versatile`, temperature `0.1`
- **UI:** Streamlit app with answer, sources, prompt, and retrieval debug panels

## Quickstart

```bash
# 1. Install dependencies
uv sync

# 2. Add your Groq API key
cp .env.example .env
# edit .env and set GROQ_API_KEY

# 3. Build indexes once
uv run python ingest.py

# 4a. Run the Streamlit UI
uv run streamlit run app.py

# 4b. Or ask from the CLI
uv run python ask.py "What margins and spacing do we use?"

# 5. Run the required validation questions and regenerate prompts.txt
uv run python scripts/evaluate.py

# 6. Build the documentation-note PDF from report/report.md
uv run python scripts/make_report_pdf.py
```

## Configuration

Main settings live in `src/fyp_rag/config.py`:

- `CHUNK_SIZE = 500`, `CHUNK_OVERLAP = 100`
- `TOP_K_DENSE = 30`, `TOP_K_BM25 = 30`, `TOP_K_FUSED = 30`
- `TOP_K_RERANK = 6` final context chunks by default
- `LIST_QUERY_TOP_K_FINAL = 10` for chapter/report-format list queries
- `SIMILARITY_THRESHOLD = 0.25`
- `REFUSAL_MESSAGE = "I don't have that in the handbook."`
- `RERANKER_MIN_SCORE = -2.0`
- `MMR_LAMBDA = 0.7`

## Validation

The six required validation queries from `docs/requirements.md` are implemented in `scripts/evaluate.py`:

1. Headings, fonts, and sizes required in the FYP report
2. Margins and spacing
3. Development FYP report chapters/sections
4. R&D-based FYP report chapters
5. Endnotes such as `Ibid.` and `op. cit.`
6. Executive Summary and Abstract

Running `uv run python scripts/evaluate.py` also tests one out-of-domain question, writes structured results to `outputs/eval.json`, and regenerates the prompt log in `prompts.txt`.

## Deliverables

- **Code:** `ingest.py`, `app.py`, `ask.py`, and the `src/fyp_rag/` package
- **Documentation note:** `report/report.md`, rendered to `report/report.pdf`
- **Prompt log:** `prompts.txt`
- **Validation output:** `outputs/eval.json`
- **Screenshots:** place Streamlit screenshots in `outputs/screenshots/` before building the PDF if they should be appended
